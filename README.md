# CloudStream MovieBox Provider

This repository contains a CloudStream extension that connects MovieBox to CloudStream, allowing you to watch movies and TV shows on your Android TV or mobile device.

## Features
- Search for movies and TV shows.
- Browse trending content.
- High-quality streaming links.
- Subtitle support.

## Installation
1. Open CloudStream 3.
2. Go to Settings > Extensions.
3. Add this repository URL: `https://raw.githubusercontent.com/[YOUR_USERNAME]/cloudstream-moviebox/master/repo.json`
4. Install the MovieBox extension.

## Development
To build the plugin locally:
```bash
./gradlew MovieBoxProvider:make
```
